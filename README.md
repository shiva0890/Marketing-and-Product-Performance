## How to Run

### Option A: Kaggle (Recommended)
1. Go to [Kaggle](https://www.kaggle.com) and create a new notebook
2. Add the dataset: *Marketing and Product Performance Dataset*
3. Upload each `.ipynb` file and run all cells

### Option B: Local
```bash
git clone <your-repo-url>
cd <repo-name>
pip install -r requirements.txt
jupyter notebook RQ1_Baseline_Performance.ipynb
```
Update `DATA_PATH` in each notebook to point to your local dataset file.
